Projeto em desenvolvimento para controle de investimentos pessoais.

O sistema está sendo construído com foco em aprendizado full stack, utilizando Node.js no backend e React no frontend.

---

## 🎯 Objetivo

Criar uma aplicação simples onde o usuário possa:

- Registrar investimentos
- Acompanhar sua carteira
- Visualizar valores investidos

---

## 🛠️ Tecnologias em estudo/uso

### Backend
- Node.js
- Express
- MySQL

### Frontend
- React
- Vite
